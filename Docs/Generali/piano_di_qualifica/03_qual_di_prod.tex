\section{Qualità di prodotto}
Per garantire la qualità del software, il gruppo adotta il modello
definito dallo standard ISO/IEC 9126. Il quadro di riferimento 
delle metriche selezionate è definito nelle \textit{Norme di progetto v1.0.0}.
