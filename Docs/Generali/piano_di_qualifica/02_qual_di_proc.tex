\section{Qualità di processo}
Per garantire la qualità dei processi, il gruppo usa la struttura definita 
dallo standard ISO/IEC 12207:1995, adattandola alle specifiche esigenze del 
progetto. Il quadro di riferimento delle metriche selezionate è definito nelle 
\textit{Norme di progetto v1.0.0}.
